次は、JavaScript のデータ型について説明します。
JavaScript には 7つのデータ型があります。例えば、テキストやナンバーなどです。
例えば、このように、テキスト型と、ナンバー型の 2 つのデータ型を宣言します。let name = “Alice”; let age = 25;
テキスト型の場合、値は、このような ダブルクォーテーション や シングルクォーテーション の中に入ります。 一方、ナンバー型は、このように単純に数値になります。
このほかにも、さまざまなデータ型がありますので、詳しくは資料を参照してください。

ここでは、データ型を確認したい変数名を渡します。例えば、この変数を渡します。let check = typeof age;

じゃ、それは、データ型と、データ型の確認の部分についてです。ここまで、皆さんは、できましたか？

次は、ジャバスクリプト内の日付（JavaScript Dates） についての部分です。 例えば、このように例をします。let d1 = new Date();　
これをコンソールログしてみると、このような書き方には、つまり、カッコの中に空白する書き方には、システムの現在の日時が返されます。

--------------------------------------------------

このようにDateに日付を渡すこともできます。let d2 = new Date(“2022-03-25”)
フォーマットは 年‐月‐日 の順になります。 時・分・秒を渡さない場合は、PCのシステムの現在のタイムゾーンが自動的に適用されます。つまり、弊社側は、このように出ます。07:00:00　
もし、システムが日本のタイムゾーン（GMT+9）に設定されている場合は、ここの出力結果は ９時（09:00:00）になります。

--------------------------------------------------

今、このように、ここに時・分・秒も追加して渡してみます。let d2 = new Date(“2022-03-25 18:30:50”);　
ページをリロードします。この時間を出力されましたね。
—------------------------------------------------
また、この渡し方以外にも、このように渡すこともできます。let d3 = new Date(2025, 8, 9, 14, 30, 5)　
つまり、年、月、日、時、分、秒 の順番で、それぞれの値を個別に渡します。 コンソールログして、この結果になります。 ここで一つ注意点があります。JavaScript では、月の値は 0 から始まります。 例えば、このように 8 を渡すと、出力される月は 9 月 になります。 別の例として、もし 0 を渡すと、let d3 = new Date(2025, 0, 9, 14, 30, 5)
結果は 1 月 として出力されます。

次は、この部分です。まずは、この部分です。 この部分とは、関数を変数に代入するということです。 先ほどのように別の関数を作る代わりに、関数を直接作って変数の値として代入することもできます。つまり、このようにします。

ここで作成した関数は、先ほどより短い構造があります。つまりこの時点では関数名はありませんね。 今、これをコンソールログしてみて、この a と b は、それぞれ 10 と 20 に置き換えられます。 ページをリロードして、返り値はこれで、正しいですね。 つまり、この関数は通常通り動作することができます。

—----------------------Đoạn 2—---------------------　
次は、このアロー関数の部分です。 アロー関数は、関数式をより短い書き方です。 例えば、このように宣言します。let sum = (a, b) => { return a + b; };　
ここでは、これは、変数を宣言しています。この部分は、アロー関数と呼ばれます。 この矢印は、functionキーワードの代わりになっています。 つまり、functionキーワードをわざわざ書く代わりに、このように矢印で書くことができます。 これをコンソールログしてみて、この a と b は、それぞれ 5 と 10 に置き換えられます。console.log(sum(5, 10));　
返した結果は、正しいですね。

—----------------------Đoạn 3—---------------------　
例えば、パラメータが1つだけがある関数の場合、カッコを使わなくても構いません。例えばこのように書けます。let square = n => { return n * n }
ここはパラメータが1つの n だけだから、カッコは書かなくても大丈夫です。 これをコンソールログしてみて、５かける５が２５になりますね。console.log(square(5));　
さらに、return が1行だけのときは、 この波カッコと return を書かなくても大丈夫です。つまり、こんな感じで書けます。let square = n => n * n 　
簡単な関数にとっては、関数式やアロー関数を使うと、書き方が柔軟で、短くなります。 それは、関数を変数に代入する部分と、アロー関数の部分です。
