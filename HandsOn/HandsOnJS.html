*Start - nếu họ hỏi strict mode là gì thì trả lời như sau, còn không hỏi thì thôi*
Strict モードは、JavaScript における厳格なモードで、"use strict";の行を書くことで有効になります。
簡単に説明すると： JavaScript がエラーをより厳密に検出するようになり、
不具合を引き起こしやすい挙動を防ぎます。例：宣言されていない変数の使用などです。
コードの安全性と可読性が向上し、 通常の JavaScript では許可されてしまう非標準な書き方を避けることができます。
Strict モードを有効にすると、 危険な動作や潜在的なエラーのある挙動の一部は完全に禁止されます。

—--------------giới thiệu—--------------
次は、JavaScriptのループです。
ループとは、同じ処理を何回も繰り返したいときに使うしくみです。
JavaScript には、いくつかの 代表的だいひょうてき なループがあります。
例えば、for ループ、while ループ、そして for...of / for...in などがあります。
今、それぞれの 特徴とくちょう を簡単に見ていきましょう。

----------------------------------------------------

—-----------ví dụ về while loop—------------
次は、whileループの例を見ていきましょう。
whileループは、繰り返す回数が分からず、ループ条件だけが分かっているときに使います。
この例を説明いたします。
let i = 0;　→　ここでは、アイという変数を宣言して、初期値を0に代入します。
while (i ＜ 3) :　これがループ条件です。各ループの前に、JavaScriptは、アイの値がこの条件を満たすかどうかを確認します。
もし条件を満たしている場合、波カッコ内のコードを実行します。アイの値を出力します。
これは、Forループと同じに、各ループ後に、i（アイ）変数が1増加します。
このループの処理流れは、Forループの流れと同じです。つまり、最初のループでは、アイの値が０になっているため、この条件を満たしていて、この結果が出ました。
次は、アイの値を１増加して、この条件と確認して、満たして、この結果を出力しました。
そのようにずっと繰り返して、条件を満たさなくなるまで、ループが止まります。
ここでは、１つの注意点があります。
この条件がずっと正しいままだと、このループは止まらずに実行され続けます。
例えば、これ (i++) を削除すると、このループは無限に実行されてしまい、そのときこのページのプログレスバーはずっと回り続けます。
そのため、この状態を避けるために、これを必ず記述する必要があります。
それは、whileループです。ここまで、皆さんできましたか？

—-----------ví dụ về vòng lặp do…while—------------
次は、do…whileループの例を見ていきましょう。
do…whileループは、whileループと似ていますが、whileループと比べて1つの違いがあります。それはdo...whileでは、条件が正しくない場合でも、波カッコ内のコードブロックが最初に実行され、その後でwhileの条件がチェックされるということです。
例えば、この例の場合は、この行の処理が実行されて、この時、アイの値は、0になっていて、このように出力します。
次は、アイの値を１増加して、この時点で、アイの値は、１になります。
その後、whileの条件を確認して、アイの値が１になっているため、この条件を満たしているため、この処理がまだ実行されて、このように出力します。
この時、またアイの値を１増加して、この時点で、アイの値は、2になります。
この時、またwhileの条件を確認して、２になっているため、満たしています。これを出力します。
またアイの値を１増加して、この時のアイの値は、３になって、whileの条件を確認して、この時のアイは、満たしていませんので、ループがここで終わります。
●例えば、アイの値を５に変更してみて、実行すると、このコードは、最初の1回目はまだ実行されます。この結果が返します。
その後、while の条件をチェックし、条件を満たさないため、それ以降は実行されません。
それは、do…whileループです。ここまで、皆さんできましたか？

次は、for…ofループの部分です。 この例を見ていきましょう。
ここでは、このような配列を宣言しています。const fruits = ["apple", "banana", "orange"];
これは、for…ofループと呼ばれます。配列の各要素を順番に取り出すために使います。

それは、for…ofループです。ここまで、皆さんできましたか？
